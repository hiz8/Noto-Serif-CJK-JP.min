Subset of the Noto Serif CJK JP.

About Noto Serif CJK

In 2014, Google and Adobe jointly developed and released Noto Sans as an open source sans serif font. It has the features of pursuing beauty, seven weights, and the ability to modify and redistribute with SIL Open Font License. Noto Serif was developed as a serif font with the same features. It is also known as "Source Han Serif" or "Genno Mincho (源ノ明朝)" in the Adobe brand.
Among the many languages, Japanese, traditional/simplified Chinese, and Korean, which have an overwhelming number of character variations, were developed separately from other languages, and were released as the "Noto Serif CJK" series. The Japanese version is "Noto Serif CJK JP".

About Noto Serif CJK JP min

Noto Serif CJK has supported a large number of Japanese characters (mainly kanji), but the file size has increased significantly. In particular, when using it as a web font, this becomes a problem and has a negative impact on page loading.
One approach that is often taken is to subset the font.
Noto Serif CJK JP min has significantly reduced the size by extracting JIS first level kanji and part of JIS second level kanji, which is not a problem for normal use.
Also, it supports the latest version 2.001 as of March 2021.
